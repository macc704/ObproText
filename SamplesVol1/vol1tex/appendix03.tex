\chapter{Javaタートル仕様}

\section{タートルの命令リスト}

\subsection{基本的な命令}

\subsubsection{rt}
\begin{description}
\item[説明]タートルを指定した角度だけ右に回す 
\item[書式]rt([角度(度)])
\item[例]rt(90);
\end{description}

\subsubsection{fd}
\begin{description}
\item[説明]タートルを指定した長さだけ進める
\item[書式]fd([長さ(ピクセル)])
\item[例]fd(100);
\end{description}

\subsubsection{bk}
\begin{description}
\item[説明]タートルを指定した長さだけ後ろに進める
\item[書式]bk([長さ（ピクセル）])
\item[例]bk(100);
\end{description}

\subsubsection{up}
\begin{description}
\item[説明]タートルのペンを上げる
\item[書式]up()
\item[例]up();
\end{description} 
  
\subsubsection{down}
\begin{description}
\item[説明]タートルのペンを下げる
\item[書式]down()
\item[例]down();
\end{description} 
    
\subsubsection{color}
\begin{description}
\item[説明]タートルのペンの色を変える
\item[書式]color([色])
\item[色の説明]java.awt.Color.[色]と指定する．色はblack,　blue,　cyan,　darkGray,　gray,　green,　lightGray,　magenta,　orange,　pink,　red,　white,　yellowから選べる．
\item[例]color(java.awt.Color.red);
\end{description} 

\subsubsection{input}
\begin{description}
\item[説明]ユーザからの数字の入力を受け付ける
\item[書式]input()
\item[例]int x = input();
\end{description} 

    
\subsubsection{print}
\begin{description}
\item[説明]文字列をコンソールに出力する
\item[書式]print([出力する文字列])
\item[例]print("こんにちは");
\end{description} 

    
\subsubsection{}
\begin{description}
\item[説明]指定した数を最大とする乱数を発生させる
\item[書式]random([数])
\item[例]int x = random(100);
\end{description}

\subsection{オブジェクトの生成に関する命令}
   
\subsubsection{new Turtle}
\begin{description}
\item[説明]タートルを生成し，名付ける
\item[書式]Turtle [タートルの名前] = new Turtle()
\item[例]Turtle taro = new Turtle();
\end{description}     
 
\subsubsection{new Object}
\begin{description}
\item[説明]オブジェクトを生成し，名付ける
\item[書式][オブジェクトのクラス名] [オブジェクトの名前] = new [オブジェクトのクラス名]()
\item[例]House taro = new House();
\end{description} 
    
\subsubsection{new ImageTurtle}
\begin{description}
\item[説明]指定された画像を初期値とする画像タートルを生成し，名付ける
\item[書式]ImageTurtle [画像タートルの名前] = new ImageTurtle("[画像のファイル名]")
\item[例]ImageTurtle taro = new ImageTurtle("image.gif");
\end{description} 
    
\subsubsection{new TextTurtle}
\begin{description}
\item[説明]指定された文字列を初期値とするテキストタートルを生成し，名付ける
\item[書式]TextTurtle [テキストタートルの名前] = new ImageTurtle("[文字列]")
\item[例]TextTurtle taro = new TextTurtle("こんにちは");
\end{description}

\section{タートルで作ってはいけないメソッド名}

タートル・グラフィックスではTutleクラスで既に定義されているメソッド名を使ってしまうと，そのメソッドの定義を上書きしてしまいます（Turtleクラスを継承しているため，メソッドのオーバーライドが行われます）．
たとえば，fd()という名前のメソッドを自分で作ってしまうと，本来Turtleに定義されているfd()メソッドが上書きされてしまい，前進することができなくなってしまいます．
そのため，以下のリストにあるメソッド名は使わないようにして下さい．

angle balance balanceX balanceY bk bounds color contains delegator die direction directionLt directionRt doubleClick down draw fd getBalanceX getBalanceY
getHeight getWidth getX getY height hide initializeLooks input inputString intersects isShow key keyDown keyPressed keyReleased keyTyped large left
leftMouseDown little location looks looksSize lt maxX maxY minX minY mouseClicked mouseDown mouseDragged mouseEntered mouseExited mouseMoved mousePressed
mouseReleased mouseX mouseY move narrow paint print random resetLooks resetScale right rightMouseDown rotate rotatedBounds rotatedHeight rotateDirection
rotatedLocation rotatedMaxX rotatedMaxY rotatedMinX rotatedMinY rotatedWidth rotatedX rotatedY rt scale scaleX scaleY setRandomSeed show size sleep small
start tall up update warp wide width x y
